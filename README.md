# Flatbrowser

Minimal Next.js app scaffold for the Flatbrowser project.

## Development

Run the local server:

```bash
pnpm dev
```

Then open `http://localhost:3000`.

## Project Structure

- `app/layout.tsx`: root layout and metadata
- `app/page.tsx`: landing page placeholder
- `app/globals.css`: global Tailwind import and base styles

## Notes

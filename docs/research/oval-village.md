## Tags to Remove or Correct

- **age: "30-40" → "18-30"** — Census 2021 data for Oval ward shows 20-29 as the dominant age group (4,318 residents, nearly double the 30-39 group at 2,339). The development's purchase-led profile may attract slightly older residents, but the area-level Census data does not substantiate "30-40" as the primary age band.
- **Description correction needed** — The .ts file preview and desc mention "canal-side" repeatedly, but no canal was found adjacent to the development. This factual claim in the prose should be reviewed.

## Summary

Oval Village is a well-verified Zone 1 development with accurate tags for zone, tenure, price, status, community, demographic, and place type. The main corrections needed are: (1) the age mix tag should shift from "30-40" to "18-30" based on Census 2021 ward data showing the 20-29 cohort as dominant, and (2) the "canal-side" description in the prose appears to be inaccurate — no canal is adjacent to the site. The character tag is correctly empty. Connectivity is strong at an estimated 17-minute average to the four reference hubs.

## Confidence: Medium-High

Zone, tenure, price, status, and demographic tags are well-supported by multiple sources. Age tag confidence is high for the correction (Census data is clear). The "canal-side" claim in the prose warrants further investigation — it is possible there is a minor waterway not surfaced in searches, but it does not appear in any developer materials, planning documents, or site descriptions found.
